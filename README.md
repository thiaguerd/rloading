# Rloading

![demo](https://media.giphy.com/media/3oz8xEyHw8fJkfd90k/giphy.gif)

## Installation

on Gemfile

```ruby
gem 'rloading'
```

on terminal

````bash
bundle install
````

application.js

````javascript
//= require rloading
````

application.css

````css
 *= require rloading
````
## Usage

Start loading

````javascript
lon();
````

Finish loading

````javascript
loff();
````
